# AI for Grievance Redressal in Public Governance

This project is an AI-powered prototype that automatically analyzes citizen grievances, classifies them, assigns priority, and routes them to the appropriate government department.

## Features
- Complaint classification using NLP rules
- Priority assignment (High / Medium / Low)
- Automatic department routing
- Simple Streamlit-based interface

## Tech Stack
- Python
- Streamlit
- Rule-based NLP

## How to Run
1. Install requirements
2. Run `streamlit run app.py`
